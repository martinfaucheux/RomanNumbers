

---- How does Roman Number work ----

1 = I
2 = II
3 = III
4 = IV
5 = V
6 = VI
7 = VII
8 = VIII


Possible characters:

I	= 1
V	= 5
X	= 10
L	= 50
C	= 100
D	= 500
M	= 1000

start at 1
stop at 3999


Example:

39 = 30 + 9 = XXX + IX = XXXIX.
246 = 200 + 40 + 6 = CC + XL + VI = CCXLVI.
789 = 700 + 80 + 9 = DCC + LXXX + IX = DCCLXXXIX.
2,421 = 2000 + 400 + 20 + 1 = MM + CD + XX + I = MMCDXXI
